# AAAI 2026 - Dataset Development Details

This folder contains the original Sampler and Scene Parameters files for the creation of the dataset used in the AAAI 2026 study.  However, due to change to the core of the Blockies code, they may not run without error.  These files can instead be reviewed to compare the distributions used for creating the different datasets.

See the corresponding TOML file to see which Samplers were used for the corresponding datasets.

To see examples of the current generation of the Blockies generator, review the notebook `examples/GenerateData.ipynb`
